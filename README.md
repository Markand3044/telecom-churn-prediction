# Telecom Churn Prediction 📊

## 📌 Project Overview

This project is about predicting whether a telecom customer will leave the company or not.

The goal is to help the business:

* Identify customers likely to churn
* Take actions to retain them
* Improve customer satisfaction

This project is based on a real business case. 

---

## 📂 Dataset

The dataset contains customer information such as:

* Call made (day, evening, night)
* Customer service calls
* Plans (international, voicemail)
* Charges
* Churn status

---

## ⚙️ Steps Done

1. Data cleaning
2. Exploratory Data Analysis (EDA)
3. Feature selection
4. Model building (Random Forest)
5. Model evaluation

---

## 🧠 Model Used

* Random Forest Classifier

---

## 📈 Output

The model predicts:

* 1 → Customer will churn
* 0 → Customer will not churn
